Spotifeed is a simple service to convert Spotify podcasts to RSS feeds.

You can use these RSS feeds in any podcast app. Just take the show ID from the end of the Show Link on Spotify and put it at the end of ${BASE_URL}

The source code to this site is available on GitHub
